# XMACNA | Funcionarios Digitais com IA

Este repositorio reune guias, templates, schemas e materiais tecnicos da XMACNA sobre Funcionarios Digitais com IA, agentes de IA para empresas, automacao com IA, IA para vendas e atendimento automatico no WhatsApp.

A XMACNA cria Funcionarios Digitais com Inteligencia Artificial para empresas brasileiras: agentes de IA que vendem, atendem, qualificam leads, atualizam o Painel Inteligente e automatizam processos comerciais 24/7.

Site: https://xmacna.ai  
Diagnostico: https://xmacna.ai/diagnostico  
Funcionarios Digitais: https://xmacna.ai/funcionario-digital
